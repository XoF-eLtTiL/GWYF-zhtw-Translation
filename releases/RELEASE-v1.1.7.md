# GWYF zh-TW Translation Pack v1.1.7

## 更新內容

- TMP 字體改為直接修補遊戲內建 `ChineseFont` FontAsset
- 移除 TMP Outline / 逐文字材質修正流程，降低連續報錯風險
- Logo 替換目標固定為 `GWYF_LOGO_1280x720`
- 修正材質沒有 `_MainTex` 時產生大量 Unity error log 的問題
- 更新繁體中文翻譯文字
- 內嵌新版 `notosanstc-medium sdf` 字體 bundle

## 安裝

將壓縮檔內容解壓到遊戲根目錄。

## 注意

- 已關閉 XUnity 自動翻譯 endpoint
- 已關閉 XUnity TMP 字體覆蓋
- BepInEx Console 視窗預設關閉
